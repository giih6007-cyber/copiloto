# copiloto
# 📚 Banco de Prompts - Copiloto

Este repositório contém o Banco de Prompts estruturado para o desenvolvimento de novas features no projeto, utilizando a Inteligência Artificial como copiloto.

## 📋 Prompts Estruturados

| Nome | Prompts | Categoria |
| :--- | :--- | :--- |
| **01 - Inicialização e Estrutura** | Crie a estrutura de pastas para um backend em TypeScript moderno focado em uma API de Copiloto, incluindo os comandos de instalação das dependências como express, typescript e vitest. | `Estrutura` |
| **02 - Arquivos de Configuração** | Gere o arquivo package.json com os scripts de dev (tsx watch) e o arquivo tsconfig.json com uma configuração moderna para Node + ESM. | `Configuração` |
| **03 - Inicialização do Servidor** | Escreva o código do arquivo src/app.ts usando express.json() e o src/index.ts configurado para rodar na porta 3000 exibindo a mensagem de Copiloto Ativo. | `Código` |
| **04 - Rota de Status do Copiloto** | Desenvolva o status.controller.ts para retornar um JSON com a mensagem 'Copiloto Ativo' e configure o arquivo de rotas src/routes/index.ts. | `Código` |
| **05 - Padronização do Código** | Configure os arquivos .eslintrc.cjs e .prettierrc para garantir que o código do projeto do copiloto siga as boas práticas de formatação. | `Qualidade` |
